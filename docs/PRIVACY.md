# Privacy Policy (Template)

This service processes uploaded CSV files for cleaning and visualization.

## Data Collected

- Account identity data from Supabase Auth (email and user ID).
- Billing metadata from Stripe (customer/subscription/payment status).
- Entitlement usage counters in the app database.

## Data Usage

- Provide access to free and paid product features.
- Enforce plan limits and billing status.
- Improve service reliability and prevent abuse.

## Data Sharing

- Supabase (authentication)
- Stripe (payments)
- Hosting/database providers used to operate the service
